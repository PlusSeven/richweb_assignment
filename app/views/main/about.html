This web application is a rich web assignment. It provides courses about data structures to learn and forums that make the users cooperate with each other.

Learning

Learning is to learn the courses about data structures with sample code and animations

  • Choose one course from the course list to learn.

  • Read the different sections of the course.

  • Play with the animation to have more idea about the course.

  • Add or update the notes if necessary.

  • Ask and answer questions for the courses in the forums with the link of the Q&A forum button.

    • Forums

    The Forums can make the users coorperate with each other.

  • The forums contain questions for each relevant courses.

  • The user can ask questions as well as answer the available questions.

  • The user can vote up each question and answer, but each user can only vote up the question or answer once.

  • The author of the questions or answers can edit his works.

  • The user can find all of the questions or answers in "My Questions" or "My Answers".
